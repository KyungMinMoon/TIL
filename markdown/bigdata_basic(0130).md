조건부 확률 (Conditional Probability) : 어떤사건이 일어난 또는 일어날 조건에서 변화된 표본공간에서 어떤 사건이 일어날 확률

```
𝑷(𝑨∩𝑩) =𝑷 𝑩 ∗ 𝑷(𝑨 | 𝑩) = 𝑷 𝑨 ∗ 𝑷(𝑩 | 𝑨)
```
