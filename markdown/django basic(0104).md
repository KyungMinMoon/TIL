## django programming

step

1) HTTP Request -> 2) URLS(urls.pt) -> 3) View(controller)
   ->4) HTTP Response(HTML)
   ->4) Model(DB)
   ->4) Template(<filename>.html)

작업의 단위는 "함수" 이다.
그래서 작업을 할 때에는, 함수를 만들어야 해!

```
def hihihi(request):
    print('hi')

urlpatterns = [
    path('admin/', admin.site.urls),
    path('wow/', hihihi)
]
```

wow라는 url이 들어오면, hihihi라는 함수를 실행시켜라.
